# Changelog

## [1.0.0] - 2026-06-08

Initial release. The following skills are included:

### Skills
- **arize-trace** — export and inspect traces and spans using the `ax` CLI
- **arize-dataset** — create, manage, and query Arize datasets and examples
- **arize-experiment** — create, run, and analyze experiments for model evaluation
- **arize-evaluator** — LLM-as-judge and code evaluator workflows
- **arize-instrumentation** — add Arize AX tracing to an LLM application
- **arize-annotation** — annotation configs and human review queue workflows
- **arize-ai-provider-integration** — manage LLM provider credentials for Arize features
- **arize-admin** — user, org, space, project, and API key management
- **arize-compliance-audit** — audit AI agents for EU AI Act, GDPR, HIPAA, and other frameworks
- **arize-prompt-optimization** — optimize prompts using production trace data and evaluations
- **arize-link** — generate deep links to Arize UI resources
- **arize-prompts** — prompt management workflows

### Install
```bash
npx skills add Arize-ai/arize-skills
```
